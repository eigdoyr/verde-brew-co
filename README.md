# Verde Brew Co. 🌿

A sustainable coffee shop single-page website built with React, TypeScript, and Vite.

## Preview

![Verde Brew Co. Preview](preview.png)

## Live Demo

[verde-brew-co.vercel.app](https://verde-brew-co.vercel.app)

## Built With

- React 19
- TypeScript
- Vite
- Framer Motion
- CSS (no framework)

## Features

- Fully responsive — desktop, tablet, mobile
- Framer Motion scroll animations
- Background color shift on scroll
- Swipeable menu carousel on mobile
- Hamburger menu with fullscreen overlay
- Smooth scroll navigation
- Lazy loading images
- SEO meta tags

## Getting Started

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
